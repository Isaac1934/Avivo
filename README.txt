
# Aivio – AI SaaS Landing Page

Thank you for purchasing the Aivio HTML Template.

## Getting Started
1. Open `index.html` in your browser to see the landing page.
2. Edit `index.html`, `css/style.css`, or `js/script.js` to customize.
3. You can host it using GitHub Pages or any static hosting service.

## Folder Structure
- index.html – main landing page
- css/style.css – custom styles
- js/script.js – custom scripts
- documentation/ – this help file
